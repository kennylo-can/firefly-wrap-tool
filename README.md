# Firefly Wrap Designer

车身贴花/拉花 3D 设计工具。加载 GLB 模型，导入 PNG/SVG 贴花，在 3D 车身上自由拖动摆放，车身改色，导出平面图。

## 快速开始

打开 `index.html`，模型自动加载。不需要拖文件。

## 功能

### 贴花（Decal）
- 导入 PNG/SVG/JPG，自动加入贴花库
- 点击车身放置，拖动移位，缩放旋转
- 跨面板拼缝不断开
- 图层排序、显隐控制

### 涂装（Paint）
- 车身颜色取色器，只染车漆不碰塑料件/装饰条
- 材质预设（哑光/金属/清漆等）
- 拾取面板模式：点选单个面板独立上色

### 视图（View）
- 爆炸图滑块，展开面板查看遮挡区域
- 仅车身分解：隐藏轮子/玻璃/内饰，车壳面板散开
- 六个视角预设按钮（前/后/左/右/顶/透视）

### 导出
- 五个正交视图独立导出 PNG（1024/2048/4096）
- 当前透视视图导出
- URL 分享：场景状态编码在链接里

## 操作

| 操作 | 方式 |
|---|---|
| 旋转视角 | 鼠标拖拽（查看模式） |
| 导入贴花 | 工具栏「＋ 导入贴花」 |
| 放置贴花 | 先点库中贴花，切到编辑模式，点车身 |
| 移动贴花 | 选中后拖拽 |
| 缩放/旋转 | 右侧滑块 |
| 删除贴花 | 选中后按 Delete |
| 改车身颜色 | 切换到涂装 tab，选颜色 |
| 爆炸视图 | 视图 tab 滑块 |
| 撤销/重做 | Ctrl+Z / Ctrl+Shift+Z |
| 导出 | 工具栏「📥 导出视图」 |

## 文件结构

```
├── index.html       # 整个应用
├── firefly.glb      # 3D 模型（自动加载）
├── export_from_blender.py  # Blender 导出脚本
├── AGENTS.md        # 开发者文档
└── README.md
```

## 导出模型

如需更换模型，在 Blender 图形界面运行 `export_from_blender.py`（Scripting 工作区 → Run Script）。输出在 .blend 同目录的 `firefly_wrap_export/`。

## 技术

纯静态页面，Three.js 0.160 从 CDN 加载。无构建步骤，无后端。GitHub Pages 直接跑。
