# Firefly Decal Designer

车身贴花/拉花 3D 放置工具 — 导入 PNG 贴花，在 3D 车身上自由拖动、缩放、旋转，实时渲染效果。

## 它能做什么

- 🖼️ **导入 PNG 贴花** — 拖入透明 PNG，自动加入贴花库
- 🖱️ **点击放置** — 选好贴花，点击车身任意位置即可放置
- ✋ **拖动移位** — 选中贴花直接拖，自动贴合车身曲面
- 📐 **缩放旋转** — 右侧面板调节宽度、高度和旋转角度
- 📦 **导出平面图** — 正前/正后/左/右/顶部五个正交视图导出为 PNG

## 快速开始

### 第一步：导出 GLB 模型（只需一次）

在 Blender 图形界面中运行 `export_from_blender.py`：
1. Blender 打开你的 `.blend`
2. 切到 Scripting 工作区 → 打开脚本 → Run Script
3. 输出在 `.blend` 同目录的 `firefly_wrap_export/`

### 第二步：打开工具开始设计

1. 浏览器打开 `index.html`
2. 拖入 `firefly.glb` 加载模型
3. 点击「＋ 导入贴花」导入你的 PNG 素材
4. 点击车身放置 → 拖动调整位置 → 右侧面板调大小
5. 点击「📥 导出视图」保存平面图

## 操作速查

| 操作 | 方式 |
|---|---|
| 旋转视角 | 鼠标拖拽空白区域 |
| 缩放视角 | 滚轮 |
| 导入贴花 | 点击「＋ 导入贴花」 |
| 放置贴花 | 先点库中贴花选中，再点车身 |
| 选中贴花 | 点击车身上的贴花 |
| 拖动贴花 | 选中后按住拖拽 |
| 调整大小 | 右侧滑块 |
| 删除贴花 | 选中后按 Delete 或点删除按钮 |
| 取消选择 | 点空白区域或按 Esc |
| 导出视图 | 点击「📥 导出视图」|
